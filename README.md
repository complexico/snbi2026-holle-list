## Overview

The repository for the source Quarto document, interweaving texts and R codes, as well as the data files for the writing of a proceeding paper titled “The Digitised Holle List Project: Building a Database from Legacy Materials for Conserving Indigenous Indonesian Languages” to be presented at [_Seminar Nasional Bahasa Ibu (SNBI) XVIII 2026_](https://snbi2025.my.canva.site/).

The information of the digitised Holle List project (and further references) can be accessed from [this page](https://portal.sds.ox.ac.uk/projects/Digitised_Holle_List/259172) and its [GitHub organisation](https://github.com/Holle-List).

The earlier work on the digitised Holle List was supported by a grant to build [lexical resources](https://enggano.ling-phil.ox.ac.uk/static/dictionary.html) for the [Enggano](https://enggano.ling-phil.ox.ac.uk/) language funded by the [Arts and Humanities Research Council](https://gtr.ukri.org/projects?ref=AH/W007290/1), UK. It was awarded to the Faculty of Linguistics, Philology and Phonetics, University of Oxford, UK as the lead institution.
